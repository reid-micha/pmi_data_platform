北極星約束 North-star constraints

「新增第 N 個之物，不需要改前 N–1 個之物。」
"Adding the Nth thing must not require touching the previous N–1 things."

  • 新 source（Kalshi / WS / chain）→ drop pmi-ingest/pollers/<name>.py，不動 pmi-core — new data source = one poller file
  • 新 PMI index → drop pmi-core/index_defs/<id>.yaml，自動 register — new index = one YAML, auto-registered
  • 新 factor → YAML 加一行 + 一個 prompt 檔，不動 column — new factor = one line + one prompt, no schema change
  • 新 prompt 版本 → bump -vN，舊 evaluation 仍指向舊 prompt（append-only） — prompts are append-only, versioned

架構速覽（給一般人） How it works · plain-language overview

一句話：把預測市場上「成千上萬個單一賭注」變成「一個你能追蹤的主題分數」。 In one line: turn thousands of individual bets into one trackable themed score.

1
🎲

預測市場 Prediction markets

成千上萬人對「某件事會不會發生」下真錢注，每個合約的價格 = 群眾估的機率。

People bet real money on whether events happen; each market's price is the crowd's odds.

2
📥

我們收資料 We collect prices

每 5 分鐘自動把相關合約的最新價格抓回來，存進資料庫並留下歷史。

Every 5 minutes we pull the latest prices for relevant markets and store the history.

3
🤖

AI 逐一判讀 AI reads each market

AI 讀懂每個市場在問什麼、方向對不對、該佔多少權重，打分並留下理由。

AI judges what each market asks, its direction and weight, then scores it with a reason.

4
📊

合成一個指數 One index number

把一組相關市場聚合成單一可追蹤分數（如「戰爭風險指數」0–100），每個數字都可回溯來源。

A group of markets becomes one trackable score (e.g. a "war-risk index"), fully traceable.

關鍵設計決策 Key invariants

  • D1 邊是 (type, weight, direction, confidence)，不是單一 scalar — typed edges, not a single similarity score
  • D3 三個量分清楚：probability(被加權值) / volume(權重) / 佔比(family normalize) — never confuse mass with magnitude
  • D5 Formula registry（具名公式），不做 eval()（RCE 雷） — named formulas, no arbitrary eval
  • D6 留在 Postgres（recursive CTE + networkx），不上 Neo4j — single Postgres, no new infra
  • D7/D8 雙迴圈解耦；edge 表空 → 退化成現況扁平行為 — graceful degradation to today's behaviour
