🟡 MVP進行中 — 現在主目標In Progress — current focus
Micah POC → Data Platform：把現有的一模一樣 migrate 到 pmi platform 上Micah POC → Data Platform: migrate the existing setup as-is onto pmi platform
  • Micah 目前已有的（maga / senate / election 指數、爬蟲、評估、UI）原封不動搬到 pmi platformMove Micah's existing maga / senate / election indexes, scrapers, evaluation and UI onto pmi platform as-is
  • 部署從 Render 搬到 AWS 生態系Migrate hosting from Render to the AWS ecosystem
  • 全鏈路真資料：爬蟲 → LLM 評估 → 產出 index → UI，沿用 Micah 邏輯、跑在新架構上Real data across the chain: scrape → LLM eval → index → UI, same logic on the new architecture
  • 三個 index 都能在畫面上看到真數字（對得上 Micah）Real numbers on screen for all three indexes, matching Micah
🔵 P1加速 + 驗證Accelerate & Validate
  • 多版本共存：改 weight/factor 會存成新版本，舊版仍能算，歷史可比Multi-version: editing weights/factors creates a new version; old versions still compute, history stays comparable
  • 多演算法框架：同一組市場可換不同算法，產出不同指數變體來比較Multi-algorithm framework: swap algorithms over the same markets to produce comparable index variants
  • 多演算法 benchmark：同一指數用多種算法回測比分，挑出最好的Multi-algorithm benchmark: backtest one index under several algorithms and pick the best
  • 一鍵回測：改完立刻看過去 90 天會跑成什麼樣One-click backtest: instantly see how the last 90 days would have run
  • 三個 category UI：maga、war、economy 各一個盤面Three category dashboards: maga, war, economy

讓「改一版 index → 立刻看歷史表現差異」變成分鐘級循環。Make "tweak an index → see the historical difference" a minutes-long loop.

🟣 P2深度訊號Deep signals
  • 廣泛驗證特徵：系統化測各種訊號／因子，找出哪些真的讓指數更準Broadly validate features: systematically test signals/factors to find which actually improve the index
  • 看「誰在交易」：從鏈上 + 訂單簿分出大戶 vs 散戶，加進指數判斷See who's trading: split whales vs retail from on-chain + orderbook and feed it into the index
  • 更聰明的 AI 評估：模糊的市場會自己上網查、讀結算條件再打分Smarter AI eval: ambiguous markets search the web and read resolution criteria before scoring
  • 耐久的工作流：長任務（回測、整表重算）跑到一半掛了能接續Durable workflows: long jobs (backtests, full recompute) resume after a crash
  • 更快的分析資料庫，等現在這套撐不住查詢再上A faster analytics DB, added once the current setup can't keep up
  • 用「關係圖」算指數：市場之間互斥／翻轉／邏輯一致，自動推出來Score via a relationship graph: mutual-exclusion / flips / logical consistency derived automatically
  • 夜間批次計算 + 成本上限，省 LLM 帳單Nightly batch compute + cost caps to cut the LLM bill
P3+平台效應Platform play

核心Core

  • 因應不同市場策略，幾分鐘就生一個新指數Spin up a new index in minutes for each market strategy
  • 分鐘級循環：定義 → 預覽 → 上線Minutes-long loop: define → preview → ship

延伸Extensions

  • AI 直接接平台：一句話拿指數、建指數AI plugs straight in: get or build an index in one sentence
  • 拖拉式建指數介面 + 回測沙盒Drag-and-drop index builder + backtest playground
  • 指數市集：第三方發布、多租戶隔離Index marketplace: third-party publishing, multi-tenant isolation
